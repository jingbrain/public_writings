# 一、WTI / Brent 在全球的地位：还有没有“同级别”的？

- 真正全球意义上的两大定价锚就是：
  - WTI（West Texas Intermediate，西得克萨斯中质油）
  - Brent（布伦特原油）
 
- 其他也很重要，但更多是区域性或组织性基准，影响力略低半级：
  - Dubai / Oman：中东→亚洲的定价锚，亚洲长约的关键参考价
  - OPEC Basket：OPEC 一揽子价格，更像是产油国“态度指数”
  - Urals（乌拉尔）：俄油对欧洲/印度出口的基准价
  - 上海原油期货（INE）：人民币计价，对亚太尤其是中国更重要

- 所以：全球媒体、金融市场和宏观分析里，WTI 和 Brent 是绝对主角，其他是配角或区域锚。

# 二、基础铺垫：这两桶油“是谁”

- 起源与地理：
    - WTI：美国内陆油，核心交割点在库欣（Cushing, Oklahoma）
    - Brent：北海海上油田，天然就是“可装船的全球油”
- 质量和物性：
    - 都是轻质低硫（Light Sweet），是炼厂的优质原料
    - WTI 更轻更“甜”，理论上应当比 Brent 贵（质量溢价）
- 代表的市场：
    - WTI 代表北美、尤其是美国内需与页岩油
    - Brent 代表欧洲 + 很大程度上的全球海运贸易（欧洲、非洲、中东→全世界）

# 三、交易机制与定价结构

- 交易所与合约：
    - WTI：NYMEX / CME，实物交割，库欣仓储能力约束很强
    - Brent：ICE，更多是现金结算/指数定价，和现实装船、现货成交挂钩
- 交割与仓储约束：
    - WTI 会出现“库欣挤兑”→ 2020 年负油价就是典型
    - Brent 受港口/航运影响更大，受单一点仓储的约束较小
- 曲线形态：
    - Contango（远期高于现货） vs Backwardation（远期低于现货）
    - 用两种曲线形态解释：市场是担心“现在缺油”还是“未来缺油”

# 四、WTI vs Brent：关系、联动和 Spread

1. Spread 定义与基本逻辑
    - Brent–WTI Spread = Brent 价格 – WTI 价格
    - 这就是“欧洲/全球海运油” vs “美国内陆油”的价差
    - 理论直觉：
        - 质量上：WTI 更好 → 有溢价
        - 现实中：运输与管道瓶颈、政策限制 → WTI 近十多年多数时间反而比 Brent 便宜
2. 驱动 Spread 的核心因素
    - 管道与基建：库欣能不能把油顺畅送到海岸线、装船出口
    - 美国出口政策：早年美国禁止出口原油，WTI 被“困在国内”折价
    - 地缘政治：中东/欧洲出事 → Brent 受冲击更大，Spread 扩大
    - 运输与保险成本：霍尔木兹海峡、红海、苏伊士运河的风险溢价
3. Spread 的“ regime ”：几种典型阶段
    - WTI > Brent：质量溢价主导、美国还没页岩油大爆发
    - WTI 深度折价：页岩油产量暴涨、基建跟不上
    - 战争/制裁期：Brent 战争溢价大幅抬升，Spread 扩大
4. Spread 的交易与宏观含义
    - 对冲/套利：做多便宜的，做空贵的（统计套利、基差交易）
    - 宏观信号：
        - Spread 扩大：美国供应相对宽松，海外/geopolitical 风险升高
        - Spread 收窄：美国出口通道畅通，全球供应更一体化

# 五、相关系数（Correlation）与联动性

- 长期相关性：  
    - 日度、周度价格历史上相关系数通常在 0.85–0.95 之间
    - 说明：它们在同一全球供需池中，只是“区域风险偏好不同”
- 短期“脱钩场景”可以写成非常有张力的故事：
    - 美国内陆事件：管道堵塞、库欣库存暴涨/暴跌 → WTI 偏离
    - 海运 & 战争事件：中东冲突、俄乌战争、红海危机 → Brent 偏离

# 六、地缘政治：为什么伊朗事件对 Brent 更致命

- 地理暴露：
    - Brent 本质上是“海运油”的代表，和中东–欧洲–亚洲的航线紧密挂钩
    - 伊朗、霍尔木兹海峡、红海、苏伊士运河一旦紧张，海运成本和风险溢价直线上升
- 美国的“护城河”：
    - 美国国内有庞大页岩油和炼厂体系，WTI 更受美国内需与政策支撑
    - 全球出事时，Brent 首先反应“全球运不运得到油”，WTI 先反应“美国够不够用”
- 叙事角度：
    - “同样一场战争，对 Brent 是正面冲击，对 WTI 是透过相关性间接冲击”

# 七、微观结构：库欣 vs 北海港口

- 库欣的“管道十字路口”角色：
    - 库欣库存数据（EIA 周度报告）如何影响 WTI
    - 存储成本、浮仓交易、金融玩家如何推波助澜
- 北海的产量衰减：
    - Brent 实体油田产量逐年下降，现在更多依赖金融合约与指数规则调整
    - “实体油在变少，但金融合约越来越多”这个矛盾可以作为一个很好的叙事点

# 八、从原油到产品：裂解价差（Crack Spread）

- 用 WTI / Brent 对应的成品油裂解价差：
    - 3-2-1 Crack Spread（3 桶原油 → 2 桶汽油 + 1 桶柴油）
    - 解释：不仅要看原油本身，还要看炼厂利润
- 区域炼厂结构：
    - 美国墨湾沿岸 vs 欧洲炼厂，对 WTI 和 Brent 的敏感度不一样
    - 某些阶段：炼厂利润高，但原油价却在跌（供需错配）

# 九、金融化：作为资产类别的 WTI / Brent

- 风险管理与对冲工具：
    - 航空公司、航运公司如何用 Brent / WTI 对冲燃料成本
    - 产油国如何用期货锁定未来收入
- 投资者视角：
    - ETF / ETN（例如 USO 等） vs 直接期货 vs 结构性产品
    - 滚动期货带来的“期货损耗”（Roll Yield），Contango 时的负收益
- 与其他资产的关系：
    - 与美元指数（DXY）的负相关性
    - 与通胀预期（Breakeven Inflation）、高收益债等风险资产的联动

# 十、宏观与政策：OPEC+、美国 SPR 与能源转型

- OPEC+ 与美国页岩油的“博弈棋局”：
    - OPEC+ 限产如何先推高 Brent，再通过相关性传导到 WTI
    - 美国作为“摇摆产油国”的角色
- 战略储备（SPR）：
    - 美国释放或补充战略储备如何对 WTI 曲线产生影响
    - 讲清“现货 vs 远期”的不同反应
- 能源转型视角：
    - 长期：新能源、电动车对石油需求天花板的讨论
    - 短期：战争 + 制裁 + 投资不足 → 经典“绿色通胀”逻辑
